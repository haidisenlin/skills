# 分阶段行动计划生成（Step 3）

---

**方法论来源：** Agile-Stage-Gate混合（Robert Cooper）· DMAIC阶段对齐（D→M+A→I→C）· CRISP-DM部署阶段 · PMI PMBOK 7th可交付物标准 · McKinsey Quick Win优先原则（90天先做不依赖大型基础设施的行动）

**前提：** 已完成调优方法论生成（Step 2）。

---

## 你的任务：将方法论分解为四阶段行动计划

### 第一步：确认信息

在输出行动计划前，请用户在飞书准备好以下内容：

⚡ **[必须调用 research-template]** 使用 read_file 读取 research-template 技能的 SKILL.md（路径见系统提示插件 Skill 列表），然后按以下参数执行：

```
topic: "工厂AI项目四阶段行动计划 - 方法论摘要与约束条件输入"
scenario: [用户已获得的调优方法论 + 推荐算法]
industry: [用户行业]
doc_name: "[厂区代码]-[项目简称]-行动计划输入-[今日日期]"
purpose: "整合方法论、约束条件和时间节点，为四阶段行动计划生成提供输入"
```

⚠️ 此步骤必须通过 read_file 读取 research-template/SKILL.md 执行，不能用飞书MCP工具或直接输出模板替代。

收到子技能返回的模板后，发给用户并说明：

```
这是行动计划输入准备步骤。

请在飞书新建文档，把以下信息整理进去：
- Step 2获得的方法论摘要（推荐算法、数据需求、改善预期）
- 约束条件和时间节点（预算、人力、管理层期望等）
- 前置条件完成状态（数据地图、GR&R、设备接口等）

完成后粘贴链接，我将生成详细的90天/6月/12月/24月四阶段行动计划。
```

**等待用户提交飞书链接或内容后，进行行动计划生成。**
